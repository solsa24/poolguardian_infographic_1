Core System Architecture

From in-pool hardware to cloud processing, this is how Pool Guardian™ operates. The system is layered for maximum efficiency and real-time responsiveness.

Hardware Layer

  • Propulsion: Dual Brushless Motors
  • Sensors: LiDAR, Optical & Thermal Cameras
  • Processing: NVIDIA Jetson Nano
  • Power: 24hr Li-ion Battery (IP68)
↓

Embedded Software

  • OS: Zephyr RTOS
  • AI Models: YOLOv7-tiny, Pose Estimation
  • Framework: TensorFlow Lite
  • Updates: Secure OTA via MQTT
↓

Cloud Infrastructure

  • Platform: AWS IoT Core
  • Storage: Amazon S3
  • Compute: AWS Lambda
  • Database: Amazon DynamoDB

On-Device AI Frameworks

The embedded AI runs a specialized suite of models, optimized by TensorFlow Lite for real-time performance on the NVIDIA Jetson Nano.

Cloud Platform Services

AWS provides a robust, scalable backend for device management, data storage, and analytics, ensuring reliable communication and control.

Communication Protocols

A multi-layered communication stack ensures Pool Guardian is always connected, whether through high-speed local Wi-Fi or cellular backup.
